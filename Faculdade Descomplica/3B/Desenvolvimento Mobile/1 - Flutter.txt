Flutter é um framework de desenvolvimento mobile criado pelo Google para construir aplicativos nativos para Android, iOS, Web e Desktop a partir de um único código-fonte. Ele usa a linguagem Dart e oferece um desempenho próximo ao nativo.

----------

Principais Características do Flutter

Código Único para Múltiplas Plataformas – Escreva uma única base de código para rodar em Android, iOS, Web e Desktop.
Hot Reload – Atualizações instantâneas no app sem recompilação completa.
Alto Desempenho – Usa o motor gráfico Skia, garantindo animações suaves a 60fps ou mais.
Widgets Personalizáveis – Tudo na interface é um widget personalizável.
Independência da Plataforma – Não usa componentes nativos; tudo é desenhado do zero.

----------

Estrutura do Flutter

Widgets → Tudo na interface é um widget (botões, textos, containers).
Material Design e Cupertino → Suporte para interfaces Android (Material) e iOS (Cupertino).
Pub.dev → Gerenciador de pacotes do Flutter.
Dart → Linguagem usada no Flutter, moderna e otimizada para desenvolvimento rápido.

----------

Componentes Principais

StatelessWidget – Interface fixa, não muda estado.
StatefulWidget – Interface dinâmica, pode mudar de estado.
Navigator – Sistema de navegação entre telas.
Provider, Riverpod, Bloc – Gerenciamento de estado.

----------

Ferramentas Necessárias para Desenvolvimento

Flutter SDK – Conjunto de ferramentas para desenvolvimento.
Dart SDK – Compilador e ferramentas da linguagem Dart.
Android Studio / VS Code – IDEs para desenvolvimento.
Emulador ou Dispositivo Físico – Para testar o aplicativo.

----------

Por que Usar Flutter?

Rápido desenvolvimento com Hot Reload.
Maior produtividade com código único para várias plataformas.
Desempenho otimizado e interface bonita.
Grande suporte da comunidade e do Google.
