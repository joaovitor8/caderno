Resumo - Arquitetura de Computadores do Século XXI

A disciplina
- Estudo da organização e funcionamento dos computadores
- Integra hardware, software e sistemas

Sistemas de numeração
- Binário (base 2), Octal (base 8), Decimal (base 10), Hexadecimal (base 16)
- Conversões entre bases → essencial em hardware e redes

Algoritmos
- Sequência de passos finitos para resolver problemas
- Propriedades: finito, definido, eficaz
- Medidas: complexidade O(n) (tempo e espaço)

Linguagens de Programação
- Alto nível (Python, Java) x baixo nível (Assembly, C)
- Paradigmas: imperativo, orientado a objetos, funcional

Abstração de dados
- Representação simplificada de estruturas
- Tipos abstratos: listas, pilhas, filas, árvores, grafos

Engenharia de Software
- Processo de desenvolvimento de sistemas
- Modelos: cascata, incremental, ágil (Scrum)
- Boas práticas: versionamento, testes, documentação

Sistemas Operacionais
- Intermediário entre hardware e usuário
- Funções: gerência de processos, memória, arquivos, dispositivos
- Exemplos: Windows, Linux, Unix

Redes de Computadores
- Comunicação entre dispositivos
- Modelos: OSI, TCP/IP
- Conceitos: IP, roteamento, protocolos de aplicação (HTTP, FTP)

Circuitos Lógicos
- Base do hardware (0 e 1)
- Portas lógicas: AND, OR, NOT, XOR
- Circuitos combinacionais e sequenciais (flip-flops, registradores)

Mineração de dados (Data Mining)
- Descoberta de padrões em grandes volumes de dados
- Técnicas: classificação, regressão, agrupamento
- Aplicações: mercado, saúde, segurança

Bancos de Dados
- Armazenamento estruturado de informações
- Modelos: relacional (SQL), NoSQL (MongoDB, Cassandra)
- Operações: CRUD (Create, Read, Update, Delete)

Segurança da Informação
- Proteção de dados contra acessos indevidos
- Princípios: Confidencialidade, Integridade, Disponibilidade (CIA)
- Ferramentas: criptografia, autenticação, firewall

Usabilidade e Experiência do Usuário
- Foco no usuário final
- Interfaces simples, acessíveis e intuitivas
- Conceitos: UX (experiência) e UI (interface)

Computação Gráfica
- Representação visual de dados
- Modelagem 2D/3D, renderização, animação
- Uso em jogos, simuladores, design

Sistemas de Informação
- Conjunto de pessoas + processos + tecnologia
- Apoio à decisão e automação de processos
- Tipos: ERP, CRM, BI

Teoria da Computação
- Estudo do que pode ou não ser computado
- Autômatos, linguagens formais, máquinas de Turing
- Classes de problemas: P, NP, NP-Completo
