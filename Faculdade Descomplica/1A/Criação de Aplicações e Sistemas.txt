Resumo - Criação de Aplicações e Sistemas

Introdução a Algoritmos
- Sequência de instruções para resolver problemas
- Deve ser: finito, definido, eficaz
- Representação: pseudocódigo, fluxograma

Tipos de Dados e Instruções Primitivas
- Tipos básicos: inteiro, real, caractere, lógico
- Instruções primitivas: atribuição, entrada, saída, cálculo

Variáveis e Constantes
- Variáveis: valores que podem mudar durante o programa
- Constantes: valores fixos, não alterados na execução

Entrada, Processamento e Saída
- Entrada: dados fornecidos ao programa
- Processamento: operações realizadas
- Saída: resultado exibido ao usuário

Estruturas de Decisão
- Permitem caminhos alternativos
- SE...ENTÃO...SENÃO
- Estruturas aninhadas e múltiplas condições

Estruturas de Repetição
- Laços para execução repetida de instruções
- Enquanto (while), Repita...até (do...while), Para (for)
- Evitar laços infinitos

Vetor
- Estrutura linear unidimensional
- Armazena vários valores do mesmo tipo
- Acesso por índice (posição)

Matriz
- Estrutura bidimensional (tabela de linhas e colunas)
- Usada para armazenar dados em grade
- Acesso por dois índices (linha, coluna)

Procedimentos sem Parâmetros
- Bloco de código reutilizável
- Executa uma tarefa, mas não recebe dados externos

Procedimentos com Parâmetros
- Recebem valores de entrada (parâmetros)
- Permitem reutilização com dados diferentes

Funções sem Parâmetros
- Semelhantes a procedimentos, mas retornam valor
- Não recebem dados externos

Funções com Parâmetros
- Recebem valores de entrada e retornam saída
- Exemplo clássico: função matemática f(x)
