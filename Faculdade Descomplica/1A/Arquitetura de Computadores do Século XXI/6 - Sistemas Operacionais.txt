O sistema operacional (SO) é o software responsável por gerenciar o hardware e fornecer uma interface para que os usuários e programas possam interagir com o computador.

----------

O Que é um Sistema Operacional?

Atua como intermediário entre o hardware e os aplicativos.

Gerencia recursos do sistema, como memória, processador, dispositivos de entrada/saída e arquivos.

----------

Principais Funções do Sistema Operacional

a) Gerenciamento de Processos
Controla a execução de programas, permitindo a execução simultânea de múltiplos processos (multitarefa).

Realiza escalonamento de processos para otimizar o uso do processador.

b) Gerenciamento de Memória
Controla o uso da memória RAM, garantindo que cada programa tenha espaço suficiente.

Usa memória virtual para expandir a RAM quando necessário.

c) Gerenciamento de Dispositivos
Controla dispositivos de entrada e saída (teclado, mouse, impressora, etc.).

Usa drivers para comunicação entre hardware e software.

d) Gerenciamento de Arquivos
Organiza e armazena dados no sistema de arquivos.

Controla permissões de acesso a arquivos e pastas.

e) Interface com o Usuário
Pode ser gráfica (GUI), como Windows e macOS, ou baseada em linha de comando (CLI), como Linux e DOS.

----------

Tipos de Sistemas Operacionais

a) Sistemas Monotarefa x Multitarefa
Monotarefa: Executa apenas um programa por vez (ex: MS-DOS).

Multitarefa: Permite a execução simultânea de vários processos (ex: Windows, Linux).

b) Sistemas Monousuário x Multiusuário
Monousuário: Suporta um único usuário por vez (ex: Windows doméstico).

Multiusuário: Permite que vários usuários acessem simultaneamente (ex: Linux em servidores).

c) Sistemas Operacionais de Tempo Real
Usados em aplicações críticas, como controle de tráfego aéreo e equipamentos médicos.

Garantem respostas rápidas e previsíveis.

----------

Principais Sistemas Operacionais

a) Windows
Desenvolvido pela Microsoft.

Interface gráfica amigável, amplamente usado em desktops.

b) Linux
Código aberto e altamente personalizável.

Usado em servidores, supercomputadores e sistemas embarcados.

c) macOS
Desenvolvido pela Apple, baseado no Unix.

Forte integração com o ecossistema Apple.

d) Android & iOS
Android: Baseado no Linux, usado na maioria dos smartphones.

iOS: Sistema exclusivo da Apple para iPhones e iPads.

----------

Modos de Execução e Núcleo (Kernel)

Modo Usuário: Executa aplicativos com acesso limitado ao hardware.

Modo Kernel: Controla os recursos do sistema com acesso total ao hardware.

Tipos de Kernel:

Monolítico: Todo o sistema roda como um único bloco (ex: Linux).

Microkernel: Módulos separados melhoram estabilidade (ex: Minix).

----------

Os sistemas operacionais são essenciais para o funcionamento dos computadores modernos, gerenciando recursos e fornecendo um ambiente para a execução de programas.
