Comandos Básicos de Seleção de Dados
  Selecionar todos os campos de uma tabela:
    SELECT * FROM nome_da_tabela;
  Filtrar dados com a cláusula `WHERE`:
    SELECT * FROM nome_da_tabela WHERE nome_da_coluna = 'valor_desejado';
  Selecionar valores distintos com `DISTINCT`:
    SELECT DISTINCT nome_da_coluna FROM nome_da_tabela;

---

Criação e Modificação de Tabelas
  Criar uma tabela de clientes:
    CREATE TABLE tabelaclientes (
      ID_Cliente INT PRIMARY KEY,
      Nome_Cliente VARCHAR(250),
      Informacoes_de_Contato VARCHAR(250)
    );

  Adicionar uma nova coluna em uma tabela (`ALTER TABLE`):
    ALTER TABLE tabelaclientes ADD Endereço_Cliente VARCHAR(250);
  Excluir uma coluna de uma tabela:
    ALTER TABLE nome_da_tabela DROP COLUMN nome_da_coluna;
  Excluir uma tabela (`DROP TABLE`):
    DROP TABLE tabelaclientes;

Criação de Relacionamentos
  Criar uma tabela de produtos com chaves estrangeiras:
    CREATE TABLE tabelaprodutos (
      ID_Produto INT PRIMARY KEY,
      Nome_do_Produto VARCHAR(250),
      Descrição TEXT,
      Categoria INT,
      Preco_de_Compra DECIMAL(10,2),
      Unidade VARCHAR(50),
      Fornecedor INT,
      Data_de_Inclusao DATE,
      FOREIGN KEY (Categoria) REFERENCES tabelacategorias(id_categoria),
      FOREIGN KEY (Fornecedor) REFERENCES tabelafornecedores(id)
    );

Filtragem de Dados
  Filtrar pedidos com valor maior que 200:
    SELECT * FROM tabelapedidos WHERE total_do_pedido > 200;
  Filtrar pedidos com valor maior ou igual a 200:
    SELECT * FROM tabelapedidos WHERE total_do_pedido >= 200;
  Filtrar pedidos menores que 200:
    SELECT * FROM tabelapedidos WHERE total_do_pedido < 200;
  Filtrar pedidos diferentes de 200:
    SELECT * FROM tabelapedidos WHERE total_do_pedido <> 200;

---

Filtragem de Textos e Datas**
  Clientes cujo nome começa com letra C ou posterior:
    SELECT * FROM tabelaclientes WHERE nome_cliente > 'C';
  Pedidos feitos a partir de 19 de setembro de 2023:
    SELECT * FROM tabelapedidos WHERE data_do_pedido > '2023-09-19';

---

Filtros Compostos
  Pedidos acima de 200 reais e com status "Pendente":
    SELECT * FROM tabelapedidos WHERE total_do_pedido >= 200 AND status = 'Pendente';
  Pedidos com status "Pendente" ou "Processando":
    SELECT * FROM tabelapedidos WHERE status = 'Pendente' OR status = 'Processando';
  Pedidos com status diferente de "Pendente":
    SELECT * FROM tabelapedidos WHERE NOT status = 'Pendente';

---

Filtros com Intervalos (`BETWEEN`)
  Pedidos com data de envio entre 1º de agosto e 1º de setembro de 2023:
    SELECT * FROM tabelapedidos WHERE data_de_envio_estimada BETWEEN '2023-08-01' AND '2023-09-01';

---

Ordenação de Dados
  Ordenar produtos pelo nome (alfabética):
    SELECT * FROM tabelaprodutos WHERE preco_de_compra BETWEEN 200 AND 600 ORDER BY nome_do_produto;
  Ordenar produtos por data de inclusão:
    SELECT * FROM tabelaprodutos WHERE preco_de_compra BETWEEN 200 AND 600 ORDER BY data_de_inclusao;
  Ordenar de forma decrescente (`DESC`):
    SELECT * FROM tabelaprodutos WHERE preco_de_compra BETWEEN 200 AND 600 ORDER BY fornecedor DESC;
